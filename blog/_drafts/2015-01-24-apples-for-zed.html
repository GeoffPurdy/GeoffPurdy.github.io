On Jan 19th, Zed Shaw aimed some pretty harsh criticism to the code school model of education in the Code Newbies podcast.
Code Newbies: The Hard Way  (code school covered at 22:45)

One specific criticism was that code school instructors do not have formal training in computer science nor teaching.

I respond to Zed's criticisms with two words:  Mason Matthews

Instructor at The Iron Yard Durham.  MS CS Duke University.  Taught at Duke and NCSSM.  Mason can calculate the asymptotic complexity of an algorithm in 200ms or less.  He can quote Edsger Dijkstra from memory and spell Dijkstra's full name correctly.

How do you like them apples, Zed?
